新的开始

Published 10-29-2017 21:39:02

不写博客已经好几年了，最近看了几篇前辈的文章自己也手痒想写点什么，框架初步定为Hexo,搭建在GitHub上，用Cloudflare 做CDN用MarkDown来写文章。

基本环境

Node.js

安装Hexo

安装Hexo

  $ npm install -g hexo-cli

创建博客

初始化博客

  $ hexo init Wasdjkl

进入文件夹

  $ cd Wasdjkl

安装依赖包

  $ npm install

配置博客

编辑配置文件 _config.yml 语言及时区填写规范

# Site
title: 知者无涯
subtitle: Wasdjkl
description: 微微风簇浪，散作满河星。
author: Wasdjkl
language: zh
timezone: Asia/Shanghai
# URL
url: http://www.wasdjkl.com

生成博客

  $ hexo generate

开启服务

  $ hexo server
